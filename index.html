📋 About This Benchmark

🎯 What We Test:

  • Limited scope for efficiency: We test 3 basic scenarios on the openclaw_sandbox benchmark (File Manipulation, Weather Data, Web Search) with 9 easy-level tasks total (3 per scenario)
  • Single-turn only: Each task is a one-shot request without multi-turn conversations, minimizing API costs and complexity
  • What empty metrics mean: When you see "—" for Accuracy or Latency, it means that model hasn't been benchmarked because of quality ranking (shown as "Pending" in Score column)

🔍 How We Select Models (From 500+ Models to Top 20):

  • Step 1 - Free Filter: From OpenRouter's 500+ models, we identify models with $0.00 pricing for both prompt and completion
  • Step 2 - Capability Filter: We exclude models that can't run text-based agent tasks:
    • ❌ Vision/image-only models (no text generation)
    • ❌ Audio/speech models (Whisper, etc.)
    • ❌ Embedding models (no conversational ability)
    • ❌ Models with <4K context (too limited for agent tasks)
  • Step 3 - Quality Ranking (like the original FreeRide): Remaining models (~15-30) are scored using weighted criteria:
    • 40% Context length (longer = better for complex tasks)
    • 30% Capabilities (function calling, structured output, etc.)
    • 20% Recency (newer models would likely to perform better)
    • 10% Provider trust (established providers like OpenAI, Meta, etc.)
  • Step 4 - Top 20 Selection: We actually run the benchmark for the top 20 highest-scoring models to balance comprehensive coverage with reasonable runtime (~2 hours daily)

📊 Composite Score Calculation:

  • 70% Accuracy (tasks passed / total tasks)
  • 20% Speed (normalized inverse latency—lower latency = higher score)
  • 10% Token efficiency (normalized inverse output tokens—fewer tokens = higher score)

💡 Understanding the Results:

  • Benchmarked models (top of list): Have full metrics including composite score, accuracy, and latency
  • "Pending" models (bottom of list, grayed out): Discovered as top-quality free models but not yet tested. Sorted by quality score. Will be benchmarked in future runs.
  • Models not shown: Either not free, excluded by capability filters, or ranked below top 20 in quality scoring
